## 前置条件

- Node.js >= 18.0.0
- DeepSeek API 密钥（在 [platform.deepseek.com](https://platform.deepseek.com/) 获取）

## 安装包

:pm-install{name="deepseek-kit zod dotenv"}

## 配置 API Key

在项目根目录创建 `.env` 文件：

```bash
DEEPSEEK_API_KEY=your_api_key
```

`createModel` 函数会自动读取 `DEEPSEEK_API_KEY` 环境变量。

或者，您也可以直接传递 API 密钥：

```ts
import { createModel } from 'deepseek-kit'

const model = createModel({
  model: 'deepseek-v4-flash',
  apiKey: 'your-api-key',
})
```

## 验证安装

创建一个简单的脚本来验证安装：

```ts
import { createModel } from 'deepseek-kit'

const model = createModel({
  model: 'deepseek-v4-flash',
  apiKey: 'your-api-key',
})
const balance = await model.balance()
console.log(balance)
```
