## 前置条件

- Node.js >= 18.0.0
- DeepSeek API 密钥（在 [platform.deepseek.com](https://platform.deepseek.com/) 获取）

## 安装包

:pm-install{name="deepseek-kit zod dotenv"}

## 配置 API Key

在项目根目录创建 `.env` 文件：

```bash
DEEPSEEK_API_KEY=your_api_key
```

`createModel` 函数会自动读取 `DEEPSEEK_API_KEY` 环境变量。

或者，你也可以直接传递 API 密钥：

```ts
import { createModel } from 'deepseek-kit'

const model = createModel({
  model: 'deepseek-v4-flash',
  apiKey: 'your-api-key',
})
```

## 验证安装

创建一个简单的脚本来验证安装：

```ts
import { createModel } from 'deepseek-kit'

const model = createModel({
  model: 'deepseek-v4-flash',
  apiKey: 'your-api-key',
})
const balance = await model.balance()
console.log(balance)
```

## 安装 Skills

Skills 是面向 AI 编码助手的领域知识包。它们能帮助你的 AI 助手更好地理解 deepseek-kit 的 API、迁移指南和最佳实践。

项目包含以下内置 Skills：

- **deepseek-kit-docs** — deepseek-kit 文档编写和管理
- **ai-sdk-migration** — Vercel AI SDK 迁移到 deepseek-kit 的指南
- **langchainjs-migration** — LangChain.js 迁移到 deepseek-kit 的指南

使用 [skills](https://github.com/vercel-labs/skills) CLI 将 deepseek-kit 的 Skills 安装到你的编码助手中：

:pm-x{command="skills add FliPPeDround/deepseek-kit"}
